SED (SEt Decimal)
=================
Affects Flags: [[D|decimal]]

| Addressing Mode  | Usage           | Hex Code | Bytes |Cycles  |
|------------------|-----------------|---------:|------:|-------:|
| Implied          |```SED```        | $F8      | 1     | 2      |

SED sets the [[decimal]] flag, which turns on decimal-encoded binary mode,
which - incidentally - the NES does not support.
