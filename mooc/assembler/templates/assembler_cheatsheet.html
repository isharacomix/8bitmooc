.bank [index]=[start] change the bank that code is compiled to .org [address] change program counter
.byte [byte] add a byte directly to the code .word [word] add a word (two bytes)
.bytes [byte1],[byte2],... add a comma seperated list of bytes to the code .ascii "[string]" add a string of ascii characters to the code
.db synonym for ".bytes" .dw synonym for ".word"
.include "[kernal]" include a kernal .incbin "[pattern]" include a binary pattern (spritesheet)
.define [label]=[value] define a constant
